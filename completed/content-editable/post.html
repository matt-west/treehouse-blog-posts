In-Browser Editing with contenteditable

HTML Element Attribute

You can define an element as editable in your HTML markup by using the contenteditable attribute. This attribute has three possible values: true, false, and inherit. Specifying inherit will make the element editable if it's immediate parent is editable.

<div id="editor" contenteditable="true">
  ...
</div>

Note: Editable elements are included in the tab order for the page which makes them accessible to users that might not be able to use a mouse. The tabindex attribute can be used to specify where in the tab order the editable element should fall.

Makings Element Editable with JavaScript

!!! IMAGE !!!

As well as the contenteditable HTML attribute it is also possible to make an element editable using JavaScript. This involves using two properties:

  • isContentEditable - This property will return true is the element is editable and false if it is not.
  • contentEditable - This property can be used to set the editable status of an element. Support values are the same of those used for the contenteditable HTML attribute: true, false, and inherit.
var editor = document.getElementById('editor');
editor.isContentEditable;
editor.contentEditable = true;

Lets take a look at an example of how you could use a button to toggle an element's editable state.

<button id="editorBtn" type="button">Enable Editing</button>
<div id="editor">
  ...
</div>

Here you've created a simple <button> and a <div>. Notice that the <div> doesn't have a contenteditable attribute. That's because you will be updating it's editable state using JavaScript.

var editorBtn = document.getElementById('editorBtn');
var element = document.getElementById('editor');

editorBtn.addEventListener('click', function(e) {
  e.preventDefault();

  if (element.isContentEditable) {
    // Disable Editing
    element.contentEditable = 'false';
    editorBtn.innerHTML = 'Enable Editing';

    // You could save any changes here.
  } else {
    element.contentEditable = 'true';
    editorBtn.innerHTML = 'Disable Editing';
  }
});

Here you start by creating two variable that represent the <button> and <div> elements. You then attach an event listener to the editorBtn that will be fired when the user clicks the button. Inside the callback function of this event listener you examine the <div> element's isContentEditable property to determine whether the the element is currently editable. If it is, you disable editing by setting the element's contentEditable property to false, and update the button text. If the element is not editable you do the reverse.

Making an Entire Page Editable

!!! IMAGE !!!

If you want to make an entire page editable you can use the document.designMode property. Setting this property to 'on' will allow you to edit the whole page. This also works on documents within iframes.

document.designMode = 'on';

Try turning on designMode for this page using your browser's dev tools. Once enabled you should be able to edit all the page content.

If you're not familiar with using your browser's dev tools, here's some steps to get you going.

  • Open up your browser's dev tools.
    • This can usually be done by right clicking anywhere on the page and selecting Inspect Element.
  • Switch to the Console tab.
  • Type in the following and press enter: document.designMode = 'on';
